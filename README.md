# Mozilla Firefox userChrome.css

## File Locations

* Windows 10: C:\Users\[username]\AppData\Roaming\Mozilla\Firefox\Profiles\[profile]\chrome\userChrome.css
* Linux/Unix: ~/.mozilla/firefox.profiles/[profile]/chrome/userChrome.css
